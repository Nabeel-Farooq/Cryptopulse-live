# 🚀 Nexus — Glass/Neon Crypto Terminal

Nexus is a sleek, futuristic crypto dashboard that blends real-time market data with a glassmorphism UI and AI-powered insights. Designed with a deep-space aesthetic and neon gradients, it delivers a fast, visually immersive way to track the crypto market.

---

## ✨ Features

### 📊 Live Market Data
- Tracks **top 50 cryptocurrencies** via CoinGecko API  
- Auto-refreshes every **60 seconds**  
- Switch between fiat currencies: **USD, EUR, GBP, JPY**

### 📈 Market Overview
- Global stats:
  - Total market cap  
  - 24h volume  
  - BTC / ETH dominance  
- Animated **marquee ticker** for live price movement

### 📋 Interactive Table
- Sortable columns (price, change, volume, etc.)  
- Embedded **7-day sparkline charts** for quick trend analysis  
- Click any coin to open a **side drawer with detailed view**

### 📉 Coin Detail View
- Live **24h area chart**  
- Expanded stats and price behavior  
- Smooth animated transitions

### 🤖 AI Forecast
- “AI Forecast” button per coin  
- Uses an edge function (Lovable AI) to generate:
  - Market outlook (**bullish / bearish**)  
  - Confidence score  
  - Key drivers  
  - Potential risks  
- Structured, fast, and contextual insights
